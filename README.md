## Estrutura

| Caminho | Descrição |
|---------|-----------|
| `2Call Cuiaba.html` / `2Call Cuiaba (producao).html` | Versões estáticas da landing page |
| `css/` · `js/` · `assets/` | Estilos, scripts e imagens da landing |
| `hello-child-2call/` | Tema-filho WordPress (base Hello Elementor) |
| `hello-child-2call.zip` | Pacote instalável do tema-filho |
| `tweaks-panel.jsx` · `js/tweaks-init.jsx` | Painel de ajustes da interface |
| `politica-de-privacidade.txt` · `TERMOS-DE-USO.txt` | Documentos legais |
| `uploads/` | Apresentação (PDF) e logo |

## Instalação do tema (WordPress)

1. Tenha o tema **Hello Elementor** instalado.
2. Em `Aparência → Temas → Adicionar novo → Enviar tema`, envie o `hello-child-2call.zip`.
3. Ative o tema-filho **Hello Child 2Call**.
